# frogger-classic
Desafio frogger classic


Funcionalidade básica -
Neste jogo você tem o jogador e os inimigos(insetos). O objetivo do jogador é
alcançar a água, sem colidir com nenhum dos inimigos. O jogador pode se mover
para a esquerda, para a direita, para cima e para baixo. Os inimigos se movem
em velocidades variáveis na parte do bloco pavimentado da cena. Uma vez que o
jogador colide com um inimigo, o jogo é reiniciado e o jogador volta para o
quadrado inicial. Quando o jogador atinge a água, o jogo é ganho.

instruções da versão do jogo implementado

Use as setas esquerda/direita do teclado para escolher o personagem,

Use a seta ↑ para selecionar o personagem.

depois de escolhido o personagem use as setas para movimentar o mesmo
o objetivo do jogo é atravessar até o lago o máximo de vezes possível

Chegar no lago vale 5 pontos e volta o personagem ao ponto inicial

Gema azul vale 15 pontos e acrescenta 1 inseto

Gema laranja vale 25 pontos e acrescenta 2 insetos

Gema verde  vale 50 pontos e cria uma rocha aleatória no caminho  

Estrela elimina metade dos insetos e todas as rochas

Rocha atrapalha a passagem do personagem

objetivo do jogo é fazer o máximo de pontos.
